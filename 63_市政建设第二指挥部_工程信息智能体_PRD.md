# 工程信息智能体

## 原需求描述

整合项目设计图纸、评估文本、政府批文、会议纪要等各类文档，构建项目专属知识库。用户可通过对话方式查询项目信息（如"清江路项目新开河河底标高是多少？"、"拼宽桥桥梁高度是多少？"、"调阅某区域施工图纸"），系统自动检索并返回精确答案，支持多轮对话和上下文理解。

---

## 页面结构

```
- 智能体首页
  - 对话入口
  - 功能导航

- 知识问答页
  - 智能问答
  - 答案展示

- 文档检索页
  - 文档库
  - 检索结果

- 图纸查阅页
  - 图纸列表
  - 图纸预览
```

## 功能列表

### 1. 文档知识库构建

- 整合文档类型：
  ```
  ├─ 设计图纸
  ├─ 评估文本
  ├─ 政府批文
  ├─ 会议纪要
  └─ 合同协议
  ```

- 智能解析入库
- 文档关联

### 2. 智能问答

- 自然语言理解
- 精准答案返回：
  ```
  问：清江路项目新开河河底标高是多少？

  答：清江路项目新开河河底标高为 3.50m（85高程）。

  来源：清江路工程施工图-排水工程图-新开河横断面图
  页码：第12页

  [查看原文] [追问]
  ```

### 3. 图纸智能检索

- 关键词检索
- 图纸定位
- 参数提取：
  ```
  问：拼宽桥桥梁高度是多少？

  答：拼宽桥桥梁高度为 8.5m。

  来源：拼宽桥设计图-立面图
  图号：Q-03
  ```

### 4. 多轮对话

- 上下文理解
- 追问跟进
- 关联问题推荐

### 5. 答案溯源

- 原文定位
- 页面跳转
- 引用标注

## 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| project_id | string | 项目ID |
| query | string | 用户问题 |
| answer | string | 答案 |
| source_doc | string | 来源文档 |
| source_page | int | 来源页码 |
